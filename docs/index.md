# Stixx Documentation

# Installation
```bash
pip install stixx
```

# Running
  - Locate the directory
```bash
$ pip show stixx | grep "Location"
```
  - Once the location has been found change directories to said location
  ```bash
  $ cd <location>
  ```
  -  Run the game
```bash
$ python game.py
```
